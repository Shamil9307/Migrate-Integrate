## Стек технологий:
* JS, HTML5, CSS, SVG, React, Redux, Axios, Node.js, Express, JWT, Bcrypt, PostgreSQL, Sequelize, React-Bootstrap

### Установка:
В директории 2 папки:
* **server/** Отвечает за back-end. 
   * cd server
   * настроить переменные окружения (см. .env.example)
   * npm i
   * npm run db
   * npm run start
* **client/** - front-end. 
   * cd client
   * настроить переменные окружения (см. .env.example)
   * npm i
   * npm run dev
